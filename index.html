VS Code 扩展挑选指南

VS Code Marketplace 收录了 50000+ 扩展，下载量百万级的「主流」扩展并不代表好用——很多是早期占位、维护停滞，或被同类新选手超越。在 AI 编辑器（Cursor / Windsurf / Continue / Cline）兴起的 2026 年，扩展选型还多了一层：要兼顾 AI 编辑器的兼容性。

本目录精选 30+ 款实战验证的扩展，按 AI 助手 / 语言支持 / Git / Markdown / 主题 / 效率 / 测试 / 数据库 / Docker / API / 远程开发 分类。每条都附 Marketplace 直链 + 多语言介绍（中 / EN / 日），并标注 「VS Code 专用 / AI 编辑器优化 / 通用」 三类兼容性标签。

不推荐的扩展我们不会列出来——这不是「全收录」列表，而是「装了就值」的精简版。打开 GitHub Copilot、Continue 或 Claude Code 的同时，你应当只装真正能加速你工作的扩展，而不是把侧栏堆满图标。

所有扩展完整列表

共 32 款精选 VS Code / Cursor / Windsurf 扩展，覆盖 AI 助手、语言支持、Git、Markdown、主题、效率、测试、数据库、Docker、API、远程开发等场景。

  1. GitHub Copilot

    GitHub 与 OpenAI 联合出品的 AI 代码补全，能根据上下文和注释自动续写整段代码。Copilot Chat 还能在编辑器里直接对话改代码、解释错误。月费 10 美元，学生和热门开源项目维护者免费。

    https://marketplace.visualstudio.com/items?itemName=GitHub.copilot

  2. Continue

    完全开源的 AI 代码助手，可以接 Claude / GPT / Gemini / 本地 Ollama 等任意模型。功能上对标 Copilot，但模型和 prompt 完全自己掌控。

    https://marketplace.visualstudio.com/items?itemName=Continue.continue

  3. Cline

    原名 Claude Dev，开源的 agent 式 AI 编辑助手，可以让模型在你授权下读写文件、跑命令。比"补全"更上一层楼，做 multi-step refactor 极快。

    https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev

  4. Codeium

    免费的 Copilot 替代品（个人用户永久免费），支持 70+ 语言。背后是同一团队做的 Windsurf 编辑器，质量比想象中好。

    https://marketplace.visualstudio.com/items?itemName=Codeium.codeium

  5. Python

    微软官方的 Python 扩展，自动捆绑 Pylance 语言服务器、调试器、Jupyter 集成。装 Python 不装它等于用记事本。

    https://marketplace.visualstudio.com/items?itemName=ms-python.python

  6. Pylance

    微软官方的 Python 语言服务器，比默认快几倍，类型检查和自动导入特别好用。装了 Python 扩展会自动一起装上。

    https://marketplace.visualstudio.com/items?itemName=ms-python.vscode-pylance

  7. Ruff

    Rust 写的 Python linter + formatter，比 flake8/black 快 10-100 倍。Astral 团队出品，已经是新项目的事实标准。

    https://marketplace.visualstudio.com/items?itemName=charliermarsh.ruff

  8. ESLint

    JavaScript / TypeScript 项目的 linting 标配。保存时自动 fix 简单错误，配 Prettier 一起用是黄金组合。

    https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint

  9. Prettier

    "代码风格不需要争论"的格式化工具，几乎所有前端项目都在用。配置极简，保存自动格式化体验丝滑。

    https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode

  10. rust-analyzer

    Rust 官方语言服务器，类型推导、跳转、refactor 全都顶级。Rust 开发体验的核心一半在它身上。

    https://marketplace.visualstudio.com/items?itemName=rust-lang.rust-analyzer

  11. Go

    Go 团队官方扩展，集成 gopls 语言服务器、dlv 调试器、go test 等所有官方工具链。

    https://marketplace.visualstudio.com/items?itemName=golang.Go

  12. Tailwind CSS IntelliSense

    用 Tailwind 的项目里写 class 时自动补全，鼠标移上去能看到展开的 CSS。提升 Tailwind 开发效率最多的一个扩展。

    https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss

  13. GitLens

    把 Git 信息深度融合进编辑器：每行代码旁边显示 blame、文件历史、commit 详情、PR 信息一目了然。免费版已经够用。

    https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens

  14. Git Graph

    在 VS Code 里看到完整的 Git commit 树状图，支持拉、推、合并、变基等所有常用操作。比命令行直观，比 SourceTree 轻量。

    https://marketplace.visualstudio.com/items?itemName=mhutchie.git-graph

  15. Markdown All in One

    一个扩展搞定 Markdown 几乎所有需求：快捷键、自动 TOC、自动列表续接、表格格式化、数学公式、导出 HTML / PDF。

    https://marketplace.visualstudio.com/items?itemName=yzhang.markdown-all-in-one

  16. markdownlint

    Markdown 文件的 linter。提示标题层级、链接、列表格式等问题，对写文档/博客非常有帮助。

    https://marketplace.visualstudio.com/items?itemName=DavidAnson.vscode-markdownlint

  17. Material Icon Theme

    把 VS Code 文件资源管理器里所有图标换成 Material Design 风格，按文件类型/语言/框架精细区分。装一次不会想换回去。

    https://marketplace.visualstudio.com/items?itemName=PKief.material-icon-theme

  18. One Dark Pro

    Atom 编辑器的标志主题移植版，是 VS Code 上最受欢迎的暗色主题之一。下载量已经过亿。

    https://marketplace.visualstudio.com/items?itemName=zhuangtongfa.Material-theme

  19. Catppuccin

    柔和、温暖的暗色主题系列（Mocha / Macchiato / Frappé / Latte 四档色调），近两年最火的主题之一。

    https://marketplace.visualstudio.com/items?itemName=Catppuccin.catppuccin-vsc

  20. Vim

    VS Code 里最完整的 Vim 模拟，支持 Ex 命令、宏、插件、leader key 配置。是从 Vim/Neovim 转过来的人的救命扩展。

    https://marketplace.visualstudio.com/items?itemName=vscodevim.vim

  21. Code Spell Checker

    代码里的拼写检查器，能识别 camelCase / snake_case，对常见技术术语友好。写英文注释和文档时尤其有用。

    https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker

  22. Path Intellisense

    写文件路径时自动补全，支持相对路径和别名（webpack/tsconfig 配的别名也认）。手动敲 ../../components/ 时会感谢它。

    https://marketplace.visualstudio.com/items?itemName=christian-kohler.path-intellisense

  23. Better Comments

    把 // TODO、// !、// ?、// * 这种特殊注释用不同颜色高亮。代码里的 TODO 一眼就能看到。

    https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments

  24. EditorConfig for VS Code

    让团队成员的 VS Code 自动遵守项目的 .editorconfig 文件（缩进、行尾、字符集等）。多人协作必装。

    https://marketplace.visualstudio.com/items?itemName=EditorConfig.EditorConfig

  25. Auto Rename Tag

    改 HTML/JSX/Vue 标签的开标签时，闭标签自动跟着改。再也不用手动同步了。

    https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag

  26. SQLite Viewer

    直接在 VS Code 里打开 .db / .sqlite 文件查看表结构和数据。无需启动 DB Browser，定位线上小问题极快。

    https://marketplace.visualstudio.com/items?itemName=qwtel.sqlite-viewer

  27. Docker

    微软官方扩展，让你在 VS Code 里管理本地的 Docker 容器、镜像、网络。Dockerfile / compose.yml 也有语法提示。

    https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-docker

  28. REST Client

    在 .http 文件里写请求直接发出去，结果显示在编辑器里。比 Postman 更轻量，请求文件可以版本控制。

    https://marketplace.visualstudio.com/items?itemName=humao.rest-client

  29. Thunder Client

    完全集成在 VS Code 里的 Postman 替代品，UI 类似 Postman 但不需要单独打开应用。免费版功能很全。

    https://marketplace.visualstudio.com/items?itemName=rangav.vscode-thunder-client

  30. Remote - SSH

    通过 SSH 连远程服务器，本地 VS Code 操作的还是远程的代码。开发服务器/EC2/树莓派的标配。

    https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-ssh

  31. Dev Containers

    把项目放进 Docker 容器里开发，团队成员的环境完全一致。配 .devcontainer/devcontainer.json 一次受用所有人。

    https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers

  32. Vitest

    Vitest 测试框架的官方扩展，可以在编辑器侧边栏看到测试树、单测点击运行。比 Jest 测试体验明显好。

    https://marketplace.visualstudio.com/items?itemName=vitest.explorer
